# Business Interpretation with Analyst Agent

## Scenario

แบบฝึกหัดนี้ให้ผู้เรียนใช้ Analyst Agent วิเคราะห์ anomaly จาก KPI
แล้วเชื่อมผลวิเคราะห์ให้เป็นทางเลือกเชิงธุรกิจที่ผู้บริหารใช้ตัดสินใจได้

## Prerequisites

1. เข้าถึง Analyst Agent ได้ในเมนู Agent
2. เตรียมไฟล์ข้อมูล KPI ตัวอย่างสำหรับแนบ
3. เลือกประเด็น anomaly ที่ต้องการวิเคราะห์ให้ชัดเจน

## Steps


## Step 1: เปิด Analyst Agent

1. จากเมนู Agent ให้เลือก All Agent
2. เลือก Analyst Agent จากรายการ และกด “Add”
3. ทำการเลือก หรืออัพโหลดข้อมูลไฟล์ Excel ตัวอย่างเข้าไปใน Prompt box


## Step 2: เลือกโจทย์ที่ต้องการวิเคราะห์
ให้ผู้เรียนเลือก 1 anomaly เช่น:

1. On‑time ลดลงที่ DC‑บางบัวทอง
2. OT พุ่งผิดปกติ   
3. เหตุการณ์เกี่ยวข้องกับความปลอดภัยเพิ่มขึ้น


## Step 3: วิเคราะห์สถานการณ์จากข้อมูล KPI

```
จากข้อมูล KPI ศูนย์กระจายสินค้าที่แนบมานี้
โดยเฉพาะประเด็น [ระบุ anomaly ที่เลือก]

ช่วยอธิบายภาพรวมสถานการณ์ในมุมผู้บริหาร:
- เกิดอะไรขึ้น
- มีนัยสำคัญอย่างไร
- ควรกังวลหรือไม่

```


## Step 4: เชื่อมผลกระทบเชิงธุรกิจ

```
เสนอทางเลือกในการแก้ไขระยะสั้น (2 สัปดาห์)
อย่างน้อย 3 ทางเลือก

สำหรับแต่ละทางเลือก กรุณาระบุ:
- สิ่งที่ต้องทำ
- ข้อดี
- ข้อจำกัดหรือความเสี่ยง
- ตัวชี้วัดความสำเร็จ (KPI)
```

## Checkpoint

- คำอธิบายสถานการณ์ตอบได้ครบ: เกิดอะไรขึ้น, มีนัยสำคัญอย่างไร, ควรกังวลหรือไม่
- ทางเลือกแก้ไขระยะสั้นมีอย่างน้อย 3 ทางเลือก
- ทุกทางเลือกมี KPI สำหรับติดตามผล

## Expected Output

- Executive interpretation ของ anomaly ที่เลือก
- ชุดทางเลือกแก้ไขพร้อมข้อดี ข้อจำกัด และ KPI
