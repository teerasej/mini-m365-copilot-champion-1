
# Copilot Work Mode 


## 1. ขั้นตอนการเข้าใช้งาน

1. เปิด link [https://m365copilot.com/](https://m365copilot.com/)
2. Login ด้วย Microsoft account ขององค์กร
3. เราจะเห็นเข้ามาที่ Copilot Chat ตามภาพ


![ภาพตัวอย่าง](../images/copilot/m365-copilot-chat.png)


## แบบฝึกหัดที่ 1: การถามถึงข้อมูลสำหรับการเปิดธุรกิจใหม่

1. เปิด link [https://m365copilot.com/](https://m365copilot.com/)
2. คัดลอกข้อความ prompt ด้านล่างไปใส่ในช่อง Chat 

ตัวอย่าง

```
สรุปหัวข้อการประชุมที่ต้องเตรียมจากไฟล์ /
```

3. เลือกไฟล์เอกสารที่เราได้อัพโหลดไว้ก่อนหน้า จาก onedrive โดยการพิมพ์ `/` ต่อจากข้อความ prompt 
   1. ใส่เป็นไฟล์ Business Idea.docx ที่เราได้อัพโหลดไว้ก่อนหน้า
4. กด enter หรือกดปุ่มส่ง (>)
5. ตรวจสอบผลลัพธ์ 


> เคล็ดลับ: ในที่นี้ถ้าเราสามารถอ้างอิงถึง meetings, บุคคลที่ติดต่อ (Contacts), หรือ Email และ Sharepoint ได้ด้วย

## ทดสอบลอง prompt เพื่อให้ Copilot เข้าไปใช้ข้อมูลจากบัญชี M365 ของเรา



1. สรุปความคืบหน้าของงานในสัปดาห์ที่แล้ว
```
สรุปความคืบหน้าของงานในสัปดาห์ที่แล้ว
```

2. มีนัดหมาย meeting อะไรบ้างในสัปดาห์นี้
```
มีนัดหมาย meeting อะไรบ้างในสัปดาห์นี้ และแนะนำไฟล์ที่สามารถใช้เตรียมตัวสำหรับ meeting เหล่านั้น
```
