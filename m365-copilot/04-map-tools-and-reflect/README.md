# Exercise: Map Tools and Reflect

## Exercise Overview

- **เวลา:** 14:45–16:00 (75 นาที)
- **เป้าหมาย:** เชื่อม Copilot capabilities กับ workflow จริง ปรับ prompt และทบทวนบทบาท Human + AI
- **ผลลัพธ์:** AI Use Canvas ฉบับปรับปรุง prompt ก่อน/หลัง และ reflection หนึ่งนาที

## Prerequisites

- AI Use Canvas จาก Day 1
- ผลงานจาก Sections 1–3 ของ Day 2
- ปากกาและ [AI Use Canvas](../../templates/ai-use-canvas.md) สำรอง

## Scenario 1: เลือกเครื่องมือให้ตรงกับงาน

Copilot แต่ละจุดเหมือนอุปกรณ์คนละชนิดในครัว: Chat ช่วยรวบรวมบริบท Word ช่วยทำเอกสาร Excel ช่วยดูข้อมูล และ Agent ช่วยทำรูปแบบงานซ้ำ การเลือกต้องเริ่มจาก step ไม่ใช่ชื่อผลิตภัณฑ์

### Practice 1: Map capability to workflow

#### ขั้นตอนการทำกิจกรรม

1. เปิด future workflow จาก Day 1 และเลือก 3 ขั้นตอนที่ AI อาจช่วยได้
2. เขียนเครื่องมือที่เหมาะข้างแต่ละขั้น: **Copilot Chat**, **Word**, **PowerPoint**, **Excel**, **Outlook** หรือ **Agent Builder**
3. สำหรับแต่ละเครื่องมือ บันทึก input, expected output, human checkpoint และ limitation
4. ตัดเครื่องมือออกจากขั้นที่ไม่มีข้อมูลที่อนุญาตหรือไม่มีผู้ตรวจผลลัพธ์
5. อัปเดตส่วน Workflow, AI Role, Data, Guardrails และ First Experiment ใน AI Use Canvas

### Practice 2: Refine one prompt and share

#### ขั้นตอนการทำกิจกรรม

1. เลือก prompt หนึ่งอันที่ใช้วันนี้และเขียนเวอร์ชันเดิมไว้
2. ปรับด้วย 5 ส่วน: **Goal**, **Context**, **Source**, **Output**, **Quality check**
3. ใช้ prompt ใหม่กับข้อมูลเดิม แล้วเปรียบเทียบความถูกต้อง ความชัด และงานแก้ไขที่ยังเหลือ
4. แชร์ภายใน 1 นาที: งานที่เลือก, Copilot ช่วยตรงไหน, Human ตรวจอะไร และ prompt เปลี่ยนอย่างไร
5. เขียนสิ่งที่จะนำไปสร้างเป็น agent ใน Day 3 หนึ่งประโยค

## Checkpoint

- ทุก mapping ผูกกับ workflow step ไม่ใช่เลือกเพราะมี feature
- prompt ใหม่ระบุ source และ quality check
- canvas มี human owner และวิธีวัด first experiment

## สิ่งที่จะได้จาก exercise นี้

AI Use Canvas ที่เชื่อม workflow กับ Copilot tools พร้อม prompt before/after และ reflection ที่นำไปต่อ Day 3 ได้

## Optional Extension

แลก prompt กับเพื่อนโดยไม่อธิบายเพิ่มเติม หากเพื่อนใช้ไม่ได้เพราะ context ขาด ให้เพิ่มเฉพาะข้อมูลที่จำเป็นและทดสอบใหม่
