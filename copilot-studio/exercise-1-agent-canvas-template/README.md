# แบบฝึกหัดที่ 1: AI Agent Canvas Template

แบบฝึกหัดนี้จะช่วยให้ทีมเปลี่ยนไอเดีย Agent ให้เป็นแผนที่ลงมือ build ได้จริง โดยใช้ AI Agent Canvas เป็นกรอบเดียวกันทั้งทีม
เมื่อจบแล้ว เราจะเห็นภาพชัดขึ้นว่า Agent นี้มีเป้าหมายอะไร, ใช้ข้อมูลจากไหน, และต้องทำ action อะไรได้บ้าง


---

## Practice 1: เตรียม Canvas Template และกำหนดโจทย์

1. ดาวน์โหลดไฟล์จากหน้ารวมลิงก์ [AI Agent Canvas Template Links](../ai-agent-canvas-template-links.md)
2. เลือก use case ที่ทีมผ่านการให้คะแนนจาก Exercise 1 แล้ว
3. กำหนดขอบเขตเบื้องต้นให้ชัดเจน โดยตอบคำถามต่อไปนี้ร่วมกัน:
   - Agent นี้ถูกสร้างมาเพื่อช่วยงานอะไร?
   - ผู้ใช้งานหลักคือใคร?
   - หลังใช้งานแล้ว ควรเห็นผลลัพธ์แบบไหน?

> **💡 Tip:** ให้ลองจำกัด use case แค่ 1 งานหลักก่อน เพื่อให้ build v1 ได้เร็วและทดสอบได้ง่าย

---

## Practice 2: เติมข้อมูลใน AI Agent Canvas

ให้ทีมช่วยกันกรอก Canvas โดยเน้นให้แต่ละช่องวัดผลได้และใช้งานได้จริง



---

## Practice 3: Review และเตรียมส่งต่อไป Module 3

1. ให้แต่ละทีมนำเสนอ Canvas แบบสั้น 3-5 นาที
2. ให้ผู้ฟังช่วย review ด้วยคำถาม 3 ข้อ:
   - เป้าหมายชัดและวัดผลได้หรือยัง?
   - ข้อมูลและสิทธิ์เข้าถึงพร้อมจริงหรือไม่?
   - Workflow และ KPI ชัดพอสำหรับเริ่ม build หรือยัง?
3. ปรับ Canvas รอบสุดท้าย แล้วสรุปให้พร้อมใช้เป็น blueprint สำหรับการสร้าง Agent v1

> **💡 Tip:** ถ้าช่อง Guardrails ยังไม่ชัด ให้ล็อกขอบเขตก่อนเริ่ม Module 3 เพื่อหลีกเลี่ยงการออกแบบ Topic ที่กว้างเกินจำเป็น
